Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • Information in the head element is used by thr browser and is not visible to the user.
  • The body element represents the visible content shown to the user.
  • Information in the body can be structured as it is visible to the user.
  • We can add images and interactive elements in the body.
  • When creating headers it is best to go in a chronological order.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS helps to bring design elements to the webpage.
  • CSS can be added to the HTML file in 3 ways: inline CSS, internal CSS and external CSS
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git helps in sharing code with team members and for version control of files.
  • git status command enables checking the branch we are on and if any files need to be committed.
  • git add commit and push commands are used to push a file from the local to the git repository.
  • It is best practice to create a feature branch when making changes and not work on the main branch.
